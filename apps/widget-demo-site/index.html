Acme Website

Это «сайт клиента» для проверки встраивания виджета поддержки. Виджет подключён обычным тегом <script> внизу страницы — ровно так, как его поставил бы реальный клиент.

Плавающая кнопка чата должна появиться в правом нижнем углу. Открой её и напиши сообщение — оно уйдёт в API (http://localhost:3000) и появится в дашборде оператора.
